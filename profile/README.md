# 👋 Welcome to NovernaLabs

**NovernaLabs** is the open-source organization behind projects developed by the Noverna team.

Our goal is to build modern, reliable, and developer-friendly software for game modding, multiplayer infrastructure, and the surrounding ecosystem. We believe in creating tools that are transparent, well-documented, and easy to contribute to.

## What we build

Our projects include:

* 🎙️ **Open Voice** - An open-source voice solution with FiveM integration.
* 🛠️ Developer tools and utilities.
* 📦 Shared libraries and SDKs.
* 🌐 Infrastructure components for multiplayer platforms.
* 🚀 Experimental projects and community-driven ideas.

## Our Philosophy

We strive to create software that is:

* Open source and community-driven.
* Well documented and easy to understand.
* Built with modern technologies.
* Focused on maintainability and long-term support.

## Contributing

Contributions of all sizes are welcome.

Whether you're fixing bugs, improving documentation, suggesting new ideas, or developing new features, we'd love to have your help.

Please check the contribution guidelines of each repository before submitting changes.
